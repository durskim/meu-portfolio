Detona Ralph

Data: Outubro de 2024 Participantes: Mariana Durski

Este projeto é um jogo simples inspirado no filme "Detona Ralph", criado pelo usuário "durskim" no GitHub. Desenvolvido para fins de estudo, o game foi construído utilizando tecnologias web fundamentais: HTML para a estrutura, CSS para a estilização visual e JavaScript para programar toda a lógica e interatividade. Trata-se de um exemplo prático de como criar um jogo básico que roda diretamente em qualquer navegador de internet.

Tecnologias: HTML CSS JAVASCRIPT
